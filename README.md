# 📝 角色提取器 (char-tiqu-baocun)

> 每轮对话后自动从正文中提取角色信息，按指定模板格式写入世界书。

## ✨ 功能特点

- **全自动提取**：每轮 AI 回复后自动分析最近对话，无需手动操作
- **智能角色识别**：通过 LLM 识别对话中出现的角色及其属性
- **标准化格式**：按你指定的模板（个人信息、背景故事、外形特征等）写入世界书
- **美观设置面板**：内嵌式 UI，与 SillyTavern 原生风格一致
- **灵活配置**：支持自定义 API、模型、温度、消息数量等
- **连接测试**：一键测试 API 是否可用
